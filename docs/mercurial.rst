Mercurial directive
===================

.. note::

    supports Python 2.7 only

.. contents:: contents
   :local:

Top commit history
------------------

::

    .. mercurial::
        :number_of_revisions: 20
        :with_ref_url:
        :include_diff:


Particular commit log
---------------------

::

    .. mercurial::
        :revision: 33:g87922kd38d8
        :with_ref_url:
        :include_diff:
